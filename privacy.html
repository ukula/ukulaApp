Privacy Policy

Last updated: 2026

Ukula is a lightweight OCR translation tool for Windows. This application does not require user accounts and does not collect personal user profiles.

Data Processing

When using translation features, selected text or captured screen regions may be processed through third-party translation providers such as:

  • Google Translate
  • DeepL
  • Microsoft Azure Translator

Translation requests are sent directly to the selected provider using the API credentials supplied by the user. Ukula does not store or transmit API keys to external servers owned by the developer.

Local Storage

Application settings such as language preferences, hotkeys, and API keys may be stored locally on the user's device for convenience.

No Analytics

Ukula does not use analytics, telemetry, advertising systems, or tracking technologies.

No Cloud Accounts

Ukula does not require account registration and does not maintain cloud-based user data.

Third-Party Services

Users are responsible for reviewing the privacy policies and terms of any third-party translation services they choose to use with Ukula.
